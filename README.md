# CollabDocs Backend

A Google Docs-inspired real-time collaborative editor backend built with Java, Spring Boot, PostgreSQL, and WebSockets.

## Tech Stack

- Java 21
- Spring Boot
- PostgreSQL
- Spring Security
- Spring WebSocket
- JPA / Hibernate

## Features

- User Authentication
- Document Management
- Real-time Collaboration
- Operational Transformation (planned)
- Version History (planned)
